操作說明

桌機滑鼠、手機觸控都順手

🖥️ 桌機

  • 滑鼠移動 環顧、瞄準（畫面中央準星）
  • 按住左鍵 蓄力，越久越大力（附拋物線預測）
  • 放開左鍵 朝準星方向丟出
  • 1 ~ 8 切換特殊彈藥（死鬥／快樂）
  • Esc 暫停 / 返回選單（可進設定）
  • M 開關背景音樂

📱 手機

  • 左下搖桿 控制視角、瞄準
  • 右下發射鈕 按住蓄力、放開丟出
  • 點右側槽 切換特殊彈藥
  • 右上 ⏸ 暫停 / 設定 / 結束
  • 自動優化 觸控裝置降畫質與陰影，省電、降溫
